# Arashi Skill Workflow Examples

This directory contains runnable walkthroughs for common Arashi tasks, grouped by difficulty.

## Example Set

- `install-first-run.md`: First-time install and verification path
- `workflow-beginner.md`: Single-repository update flow
- `workflow-intermediate.md`: Multi-repository branch alignment flow
- `workflow-advanced.md`: Recovery and synchronization flow

Each example includes prerequisites, command sequence, expected outcomes, and common failure checkpoints.
